## 🔧 Frequently Asked Questions

What if the trainer does not launch?
Check if you installed the .NET Desktop Runtime. You can find this on the official Microsoft website. Ensure your antivirus software does not block the application. You might need to add the folder to your antivirus whitelist.

Will I get banned?
Any tool that interacts with game files carries risk. Use this trainer on an account you do not mind losing. We recommend using it on secondary accounts first to understand how it functions before applying it to your main profile.

How do I update the software?
Delete the old folder entirely. Download the new version from the link provided and follow the setup instructions again.

Does the aimbot work on all killers?
The aimbot works for killers using projectile attacks. It offers limited utility for basic melee attacks. Adjust the sensitivity slider in the settings tab to match your mouse DPI settings.

Can I customize the ESP colors?
Yes. Click on the Visuals tab in the interface. You will see a list of ESP toggles. Click the color box next to each item to open a picker and assign a new color.

Does the Bloodpoint bot work while I am away?
Yes, but you should check on the progress every hour. If the game crashes or results in an error screen, the bot might stop functioning. It requires the game to stay in the menu or trial lobby to operate.

What do I do if the performance drops?
Open the Settings menu and disable the ESP modules. Visual rendering consumes more CPU resources than the automation features. Limiting the refresh rate of the overlay will also improve performance.

How do I close the program?
Press the Quit button inside the tool interface to close all active processes and clear the memory hooks safely. Avoid force-closing the application through Task Manager, as this can leave fragments in your system memory.
